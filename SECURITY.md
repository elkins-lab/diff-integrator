# Security Policy

## Supported Versions

The following versions of `diff-integrator` are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them directly to the project maintainer by sending an email to `george@example.com`.

You should receive a response within 48 hours. If the vulnerability is accepted, we will coordinate with you to publish a fix and, if appropriate, issue a CVE and security advisory giving you credit for the discovery.
